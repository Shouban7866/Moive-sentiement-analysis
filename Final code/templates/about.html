About MovieSent

MovieSent is a movie review sentiment analysis project that uses Machine Learning and Deep Learning to classify reviews as Positive or Negative. The goal of this project is to show how text data can be analyzed using AI.

How the Project Was Created

This project was built step-by-step using simple and commonly used AI techniques. Here is a short explanation of how the system was created:

1. Collecting Movie Reviews

The project started with collecting a dataset of movie reviews where each review had a sentiment label (positive or negative).

2. Cleaning and Preparing the Text

Before training any model, all reviews were cleaned using basic NLP steps such as:

  • removing HTML tags
  • converting text to lowercase
  • removing stopwords
  • tokenization
  • lemmatization

3. Training Two Different Models

To make the system powerful, two AI models were trained:

  • Logistic Regression (ML Model): trained using TF-IDF features.
  • LSTM Neural Network (Deep Learning Model): trained using token sequences.

4. Building the Web Application

After training the models, a simple and fast web application was created using Flask. The user writes a review → text is preprocessed → selected model predicts the sentiment instantly.

Benefits of This Project

  • Real-time Analysis: Users get instant sentiment predictions.
  • High Accuracy: Uses both ML & Deep Learning for better results.
  • Educational Value: Great example of how NLP and AI work together.
  • User-Friendly: Clean and easy-to-use web interface.
  • Practical Application: Can be extended to product reviews, social media posts, feedback systems, etc.
  • Lightweight Web App: Can run on any computer with Python.
  • Shows Full AI Pipeline: From data cleaning to model training to deployment.

Conclusion

MovieSent is a simple, effective, and complete AI project that demonstrates how text sentiment analysis works in real life using modern machine learning techniques.
