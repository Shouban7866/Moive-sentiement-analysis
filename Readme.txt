How to run (local):
1) Download Code/ folder from the ZIP and unzip.
2) Install dependencies:
   pip install -r requirements.txt+nltk_setup.txt
3) Dataset (download from the Drive link)
4) Place heavy model file 'lstm_model.h5' (download from the Drive link)
   in the same folder as app.py (Code/).
5) Run:
   app.py
6) Open browser: http://127.0.0.1:5000

Notes:
- model.keras (~57 MB) is NOT inside the ZIP to keep ZIP <30MB.
Goole drive link = https://drive.google.com/drive/folders/11TP44cdOEXPV0P1Ut8G6cqtF28lbt-Dc?usp=drive_link
